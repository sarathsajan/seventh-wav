Band-Split Rotary Position Embedding Transformer


BS-RoFormer is the current SOTA (Dec 2025) for splitting audio to multiple stems for the purpose of voal removal, audio mixing and isolation. In this context stems mean the different individual tracks that make up a song. They include vocal, piano, violin, drums, flute and more instruments. In this project we will use a more refined model that can be accessed via an API call provided by the MVSEP website (mvsep.com)

White paper link - https://arxiv.org/abs/2309.02612
